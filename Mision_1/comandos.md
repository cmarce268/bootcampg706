

# Comandos 
`😂` git -- version 

`cd` me sirve para navegar en carpetas
`git init` - Inicializa un repositorio vacio
`git`  - muestra el estado en el que estan los archivos
`git add . `- Este comando agrega todos los archivos
`set-executionPolicy`
`git commit -m "primer commit"` - Sella la caja (first commit indica que hasta donde queda el codigo ahi sella)

`python --version` para concoer la version de python
crear un entorno virtual en python 
`python -m venv env312`
Activar el entorno virtual `env312\scripts\activate`

Cuando salen errores se debe habilitar el powerShell `Set-ExecutionPolicy Unrestricted`  para habilitar ejecución de comandos
`pip list`
